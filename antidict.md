# Antidict frontend

Для проекта мы используем Create React App, потому что он предлагает
готовую настройку проекта, что существенно ускоряет время для создания
и запуска проекта

## Зависимости

* `react-router-dom` для создания ссылок
* `antd` элементы на странице

## Запуск проекта

```shell script
npm start
```
